# Shop Management System

## Description
This django project defines:
- shop app for managing products and their images with Django REST Framework,   using a nested API structure.
- core app for customizing user authentication by extending Django's user model with unique email constraints.

## Installation Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/arghavankb/tmn_backend_task.git
2. **Navigate to the project directory**
   ```bash
   cd tmn_backend_task
3. **Create a virtual environment for the Python project**
   ```bash
   python -m venv venv
4. **Activate the virtual environment on Ubuntu**
   ```bash
   source venv/bin/activate
5. **Install requirements**
   ```bash
   pip install -r requirements.txt
6. **Migrate the database**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
6. **Run the project**
   ```bash
   python manage.py runserver
   
## Technologies Used
- **Python**: Core language for implementing the project.
- **Django REST Framework**: Framework used for building the API.
- **MySQL**: Database used to store products and users.
